Facundia is a library for simple manipulation of single words in Spanish. There are many like it, but this one is mine.  It does not use an API or DB, does not have external dependencies, and can be included on any Java project using Maven. More info 

The Facundia library is free under Apache Software License 2.0 

Anyone can have access to the code on Github 

Contributions are more than welcome!

Jitpack  is an awesome site that allows you to release in Github and use those releases as Maven dependencies on other projects.

The badge below will allow the inclusion of any version of Facundia in your project:

  • Negative numbers
  • Plural detection
  • Infinitesimal numbers
  • Ordinal numbers
  • Verb conjugations! Although, Spanish verbs are somewhat complex , so this is going to take a while
  • Want something else? Create an issue or email me.
